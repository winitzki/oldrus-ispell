Tools for Old Russian Orthography -- Инструментар╕й для старой русской ор╛ограф╕и

Digest: This page contains tools for using the old (pre-1918) Russian orthography on a computer. A special 8-bit encoding (KOI8-C) is developed for this purpose. The tools include: a KOI8-C specification; KOI8-C keyboard maps; KOI8-C screen fonts (X Window .bdf fonts and Windows .fon fonts); a program to generate Postscript from plain text and BDF fonts; a spellchecking dictionary for ispell, a script to help converting to/from old orthography. All software on this page is free software in the sense of the Free Software Foundation.

Tools:

(Some of the linked pages are in Russian.)
  • KOI8-C documentation and tools, including:

Enjoy!
